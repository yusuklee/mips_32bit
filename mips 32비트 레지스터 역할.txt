✅ 쉽게 외우는 팁
용도	이름	번호	기억법
상수 0	$zero	$0	항상 0, 바뀌지 않음
반환값	$v0, $v1	$2, $3	Value 반환
인자	$a0 ~ $a3	$4 ~ $7	Argument
임시값	$t0 ~ $t7	$8 ~ $15	Temporary
$t8 ~ $t9	$24 ~ $25	계속 이어짐
저장값	$s0 ~ $s7	$16 ~ $23	Saved (함수 끝나도 유지)
스택 포인터	$sp	$29	Stack Pointer
프레임 포인터	$fp	$30	Frame Pointer
복귀 주소	$ra	$31	Return Address (함수 복귀용)


✅ 암기용 미니 요약표
$0   → $zero      (0)
$2~3 → $v0, $v1   (반환값)
$4~7 → $a0~$a3    (인자)
$8~15 → $t0~$t7   (임시)
$16~23 → $s0~$s7  (저장)
$24~25 → $t8~$t9  (임시 계속)
$29 → $sp         (스택)
$30 → $fp         (프레임)
$31 → $ra         (복귀)
